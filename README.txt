Sistema Web VI
==============

Projeto desenvolvido para a disciplina: laboratorio de desenvolvimento multiplataforma. Onde
o objetivo será cadastrado clientes.

Este projeto está usando as bibliotecas: Express e Handlebars. Para instalar use o comando:
      npm install

Para rodar use o comando:
      npm run dev

As rotas são:
    POST: Para cadastrar, a endpoint é / -> http://localhost:8081/cadastrar
    GET: Para listar, a endpoint é /consultar -> http://localhost:8081/consultar
    GET: Para listar, a endpoint é /consultar -> http://localhost:8081/
    POST: Parar atualizar, a endpoint é /atualizar -> http://localhost:8081/atualizar/:id
    DELETE: Parar deletar, a endpoint é /deletar -> http://localhost:8081/atualizar/:id
